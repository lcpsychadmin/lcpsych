How it works

  • Each URL is fetched via a HEAD request and its HTTP status code recorded.
  • 410 Gone — correct. Google will de-index the page permanently.
  • 404 Not Found — click + 410 list to make the server return 410 for that path automatically.
  • 200 OK — page still live. Remove or archive the content first.
  • Once a URL returns 410, submit it manually in Google Search Console → Removals for faster de-indexing.
