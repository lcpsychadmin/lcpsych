How it works

  • Each URL is submitted to the Google Search Console URL Removals API as a Temporary Hide (removes the URL from search results for ~6 months).
  • The server uses the GOOGLE_CLIENT_EMAIL service account credentials configured in environment variables.
  • URLs must be within the GSC_SITE_URL property ({{ gsc_site_url }}).
  • For permanent removal, delete or redirect the page first, then wait for Googlebot to recrawl.
